Тестовое задание WinForms 

Требования .NET FrameWork (не Core), MS SQL (можно самый простой вариант файлового), EntityFramework

Написать простое приложение с оконным интерфейсом для учета сотрудников предприятия.
Система должна позволять вводить, изменять и удалять сущности:
- подразделение
- сотрудник

Главное окно должно содержать "дерево" подразделений и список сотрудников. Список сотрудников "фильтруется" по выбранному подразделению.

"Реквизиты" подразделения:

- Наименование
- "Родительское" подразделение
- Дата создания
- Дата закрытия

"Реквизиты" сотрудника:

- ТН (табельный номер)
- Фамилия
- Имя
- Отчетство
- Пол
- Дата рождения
- Место рождения
- ИНН (налоговой номер)
- Дата приема на работу
- Дата увольнения
- Причина увольнения

Для каждого сотрудника ведется отдельный список его "рабочих подразделений". Каждая запись содержит данные:
- Дата перевода (для первого назначения должна быть равна дате приема)
- Подразделение
- Должность (профессия)

Допускается использование компонент DevExpress или любых других графических элементов, распространяемых по "открытым" лицензиям
